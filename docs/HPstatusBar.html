Class: HPstatusBar

HPstatusBar()

Represents a health points (HP) status bar. This class manages the display of the HP status bar and updates it based on the current health value. It extends the `DrawableObject` class and uses a series of images to represent different health levels.

Constructor

new HPstatusBar()

Creates an instance of `HPstatusBar` and initializes its properties. This method loads the required images and sets the initial position and size of the status bar.
