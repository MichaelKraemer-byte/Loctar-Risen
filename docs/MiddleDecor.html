Class: MiddleDecor

MiddleDecor(path, x)

Represents a middle decoration in the game. This class defines the properties and behavior of a middle decoration object that moves at a certain speed in the game world.

new MiddleDecor(path, x)

Creates a new instance of the `MiddleDecor` class and loads the image.
